This tool queries wikidata for details of memberships of parliaments.
Items for the members of parliament need to contain position held (P39) statements with all of the following qualifiers in order for the tool to work:
  • start time (P580)
  • end dates (P582) (where appropriate)
  • parliamentary term (P2937)
  • parliamentary group (P4100)
In addition, for colours to appear on the chart, the item for the parliamentary group must have an sRGB color hex triplet (P465) statement. Items without this statement will appear in black.
